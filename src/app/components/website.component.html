Here's How and Why Gauravjot.com

I, Gauravjot Garaya have created this website by myself, solely for the purpose of showcasing my skills and accomplishments. This website is created using Angular 6, Bootstrap, HTML5 and CSS. Keeping it simple and elegant is theme going into developing website's minimalistic design.


What I used & why?


  • Angular

    Version 6

    This is one of my first projects with Angular 6 and I would surely say it is an execellent framework and quicker than others like React. The structure is clean and compilation is no sweat. Surely recommended to anyone!

  • Bootstrap

    Version 3.3.7

    Bootstrap makes life easier with easy to use classes and CSS features which otherwise requires ton of time to write. Excellent project, responsive and basis of any webstie I build. An A+ !

  • HTML5 and CSS3

    Global

    HTML5 and CSS3, as general of markup and style language as it gets to build a simple website for you folks out there. Nothing much to say here.

  • Atom.io

    Text Editor

    Excellent text editor for building almost any project. It's open source and have tons of packages and add-ons for almost anything needed for this project.
